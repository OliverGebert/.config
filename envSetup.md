# Environment Targets

## Targets
- Support the development in shell, HTML, CSS, JS and Python, including LUA and Perl.
- Provide the following technology stack:
    - iTerm2
    - homebrew
    - ZSH
    - tMux
    - NeoVIM
    - Vagrant
- Additional applications:
    - Postman
    - Wireshark
    - VirtualBox
