## tMux CLI commands

| topic | command | comment |
| --- | --- | --- |
| tmux | tmux ls | show sessions |
| tmux | tmux attach | re-attach to existing session |
| tmux | tmux kill-server | kill server to ensure .tmux.conf is loaded |
